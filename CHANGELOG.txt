All notable changes to this project will be documented in this file.

and this project adheres to [Semantic Versioning](http://semver.org/spec/v2.0.0.html).

Time Format: YYYY-MM-DD

Major.Minor.Patch
Major 
	Erhebliche Änderungen im Funktionsumfang und umfangreiche Neuerungen auf Basis der Vorgänger. 
	Eine Vorgängerversion sollte mindestens einmalig erfolgreich z.B. während einer Strahlzeit eingesetzt worden sein bevor sie als Grundlage für die nächste Major dient.
Minor
	Aktualisierung mit einer abgeschlossenen Verbesserung / Neuerung kleineren Umfanges
Patch
	Bug-Fixes oder kosmetische Änderungen.
